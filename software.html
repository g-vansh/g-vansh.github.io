Tools built en route

Surveyors make their own instruments. Most of these started the same way: a question we couldn't answer until we built the tool. Everything is open source at github.com/g-vansh.

04Instruments open source, in active use

STE — Strategic Treatment Effects

R package · random forests + LASSO under the Rubin causal model

Estimates which firms benefit from a strategic choice — and which are hurt by it — from high-dimensional observational data. Built from scratch as the companion to Guzman's Treatment Effects in Managerial Strategies; propensity scores from out-of-fold random forests, local regressions for the effects, LASSO for the determinants.

Upload-to-Zenodo

Python utility · adopted by the AEA Data Editor's replication lab

Synchronizes research archives between local storage and Zenodo. Started as a side-project while running reproducibility checks for the American Economic Association; the 40-person replication team still uses it.

GitHub

The gazette-reading machine

LLM cascade · 16.4 million paragraphs and counting

The measurement engine behind the Brazil state-capacity project: a multi-stage cascade that reads municipal gazettes at scale — cheap models triage, strong models adjudicate, a silver-standard set keeps everyone honest. Code released with the papers.
